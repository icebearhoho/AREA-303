# AREA 303 — Frontend

This folder will hold the **Next.js 14 (App Router)** dashboard. It is intentionally
empty for now — see the top-level `README.md` and the `Init Next.js 14 App Router
+ TailwindCSS + shadcn/ui` item on the project roadmap.

Once initialised, the expected layout:

```
frontend/
├── app/                 # App Router routes
│   ├── (dashboard)/     # sidebar-layout group with the 17 feature links
│   └── layout.tsx
├── components/          # shared UI (shadcn)
├── lib/
├── public/
├── next.config.mjs
├── tailwind.config.ts
├── package.json
└── .env.local.example
```
